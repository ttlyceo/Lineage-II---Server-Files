IF U WANT TO ADD A NEW LANGUAGE

1.- You need to add a new Line in language_setting.xml with the name, folder and Abbreviation (Folder and Abbreviation need Just 3 letters)
2.- When u done just duplicate the eng folfer and change the name (Abbreviation)
	EXAMBPLE:
		NEW LINE IN language_setting: <language name="Spanish" folder="esp" abbreviation="esp" />
		FOLDER NAME: esp

3.- If u have any doubt u can send us an Email or asking in our Forum.
