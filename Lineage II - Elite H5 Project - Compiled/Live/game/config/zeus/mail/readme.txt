IF U WANT TO ADD A NEW LANGUAGE

1.- If u dont have any language setting u need to add a new Line in language_setting.xml with the name, folder and Abbreviation (Folder and Abbreviation need Just 3 letters)
2.- When u done just duplicate the eng folfer and change the name (Abbreviation)
	EXAMBPLE:
		NEW LINE IN language_setting: <language name="Spanish" folder="esp" abbreviation="esp" />
		FOLDER NAME: esp

3.- If u have any doubt u can send us an Email or asking in our Forum.

如果你想新增語言

1.- 如果您沒有任何語言設置，則需要在 language_setting.xml 中添加一個新行，其中包含名稱、資料夾和縮寫（資料夾和縮寫只需 3 個字母）
2.- 完成後，只需複製 eng 資料夾並更改名稱（縮寫）
範例：
language_setting 中的新行：<language name="Spanish" folder="esp" abbreviation="esp" />
資料夾名稱：esp

3.- 如果您有任何疑問，您可以給我們發送電子郵件或在我們的論壇上提問。
